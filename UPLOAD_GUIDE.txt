둥둥이 AI 판례집 간단형 v2

변경 내용
- 기존 전체 스크린샷형 이미지를 제거
- 왼쪽은 실제 HTML 텍스트 소개 영역으로 구성
- 오른쪽은 법정 장면 이미지 한 장만 사용
- 공개 판례는 제목 카드만 유지
- 최신 제1호 판례 수정본 반영

GitHub 업로드
1. ZIP 압축을 해제합니다.
2. 저장소 루트에서 Add file → Upload files를 누릅니다.
3. index.html, assets, cases, UPLOAD_GUIDE.txt를 모두 업로드합니다.
4. 기존 파일을 덮어쓴 뒤 Commit changes를 누릅니다.
