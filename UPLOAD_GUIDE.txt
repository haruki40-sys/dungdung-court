둥둥이 AI 판례집 간단형 GitHub 업로드 패키지

변경 내용
- 메인 화면을 간단한 형태로 전면 정리
- 대표 이미지는 dungdung-court-main.webp 하나만 사용
- 공개 판례 카드는 제목만 표시
- 기존 소개 문구, 진행 방식, 판례 요약, 배지 등 제거
- 기존 판례 상세 페이지 2개는 그대로 유지

GitHub 업로드 방법
1. ZIP 파일을 압축 해제합니다.
2. 저장소 루트에서 Add file → Upload files를 누릅니다.
3. 압축 해제한 폴더 안의 index.html, assets, cases, UPLOAD_GUIDE.txt를 모두 올립니다.
4. 기존 파일 덮어쓰기를 허용한 뒤 Commit changes를 누릅니다.
