반영 내용
1. 제2호 판례 추가
2. 메인 공개 판례 목록에 제2호 카드 추가
3. AI 배심원은 검사·변호사 공방을 독립적으로 검토하고,
   인간 의견은 별도 참고 기록으로 남긴다는 소개 문구 반영

GitHub 업로드 방법
1. ZIP을 압축 해제합니다.
2. 저장소 루트에서 Add file → Upload files를 누릅니다.
3. 압축 해제한 폴더 안의 index.html, assets, cases, UPLOAD_GUIDE.txt를 모두 업로드합니다.
4. 기존 파일 덮어쓰기를 허용하고 Commit changes를 누릅니다.
5. GitHub Pages가 자동 재배포될 때까지 잠시 기다립니다.
