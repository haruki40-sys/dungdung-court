변경 내용
- 홈 상단의 큰 흰색 제목/부제 영역 제거
- 홈 대표 이미지를 고해상도 신규 법정 일러스트로 교체

GitHub 업로드
1. ZIP 압축을 해제합니다.
2. 저장소 루트에서 Add file → Upload files를 누릅니다.
3. index.html, assets, cases, UPLOAD_GUIDE.txt를 모두 업로드합니다.
4. 기존 파일을 덮어쓴 뒤 Commit changes를 누릅니다.
